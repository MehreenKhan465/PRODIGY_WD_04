# PRODIGY_WD_04
Personal Portfolio Website — Task-04

A clean, modern, and responsive Personal Portfolio Website built using HTML, CSS, and JavaScript. This portfolio is designed to showcase skills, projects, and achievements in a visually appealing Peach & Cream theme. It includes smooth navigation, scroll-based navbar effects, project cards, and structured sections for About, Skills, Projects, and Contact.<br>

Features

• Fully responsive portfolio
• Smooth navbar color change on scroll
• Clean and modern Peach & Cream UI theme
• Home section with headline, intro, and call-to-action button
• About Me section with bio, education, and goals
• Skills section with interactive skill tags
• Projects grid layout with cards
• Contact section with email, phone, LinkedIn, GitHub
• Footer with copyright
• Simple and clean JavaScript animations (scroll detection)<br>

Tech Stack

• HTML5
• CSS3
• JavaScript (ES6)<br>
